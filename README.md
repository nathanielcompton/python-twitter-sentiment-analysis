# Twitter Sentiment Analysis

Leveraging Python's `tweepy` and `textblob` libraries.

### Installation and Usage
```
$ git clone <repo URL>
$ cd <path to repo>
$ python3 -m venv venv
$ python3 source venv/bin/activate
<venv>$ pip3 install -r requirements.txt
<venv>$ python3 app/analyzer.py
```

And away we go!
